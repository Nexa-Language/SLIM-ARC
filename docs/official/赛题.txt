内存受限环境的大语言模型推理优化问题	Runtime Optimization of LLM Inference for the Memory Constraint System

A	学术型

赛题背景
大语言模型相关技术发展突飞猛进，但同时模型海量的参数和运行时的大量KV　cache产生了大量的内存资源需求，使得资源受限场景难以部署。但是，大语言模型推理过程中访存行为固定，而且专家激活数量较少，为使用虚拟内存、数据预取等相关技术提供了潜在的施展空间，使得压缩运行时内存成为可能

赛题任务
分析LLM推理过程的访存行为分析，包括但不限于参数加载、KV cache使用、专家激活等。
使用虚拟内存的相关技术，通过按需加载、数据换出等策略减少运行时所需的物理内存量。
通过预取等技术掩盖IO延迟，提升推理性能。
赛题特征
本课题注重在嵌入式设备、边缘计算设备上的大模型部署。
强调对学生的全局系统观念的考察，特别是数据访问、传输、硬件特性、计算精度等多重因素相互影响的复杂场景的处理。
参考资料
FlexInfer: Breaking Memory Constraint via Flexible and Efficient Offloading for On-Device LLM Inference (EuroMLSys 2025)
